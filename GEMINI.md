# Project Overview: Resumes

This repository contains the professional resumes of **Muhammad Saqib**, a Software Engineer with over 6 years of experience. The resumes are maintained in Markdown format and are tailored for different roles, highlighting specific technical strengths and career focuses.

## Directory Overview

This is a **Non-Code Project** serving as a central repository for personal career documentation. It allows for version-controlled management of resume variations tailored to specific job markets (Frontend, Full Stack, General Software Engineering).

### Key Files

- **[resume.md](./resume.md)**: The general-purpose resume. It provides a balanced view of full-stack capabilities, including Frontend (React, Elm, Angular), Backend (Node.js, Golang), and System Architecture.
- **[resume_fe.md](./resume_fe.md)**: A specialized resume tailored for **Senior Frontend Engineer** roles, with a heavy emphasis on the React ecosystem, Design Systems (Paack-UI), performance tuning, and frontend leadership/mentorship.
- **[resume_mern.md](./resume_mern.md)**: A specialized resume tailored for **Senior Full Stack Engineer** roles, focusing on the MERN stack (React, Node.js, Express.js) and database management (PostgreSQL, MongoDB).

## Usage

These files are intended to be:
1.  **Updated:** Periodically as new skills are acquired or projects completed.
2.  **Exported:** Converted to PDF or other formats for job applications using tools like `pandoc` or Markdown-to-PDF extensions.
3.  **Referenced:** Used as a base context for AI interactions to generate cover letters or prepare for interviews.
